Comment 1 Wayne Rasband 2015-02-17 19:47:19 CST 
Using an image from the File>Open Samples menu, please provide a list of steps that reproduce this problem. Something like the following example, which works as expected.

  1. Open Clown sample image.
  2. Using the Oval selection tool (second tool in tool bar), outline the clown's left eye.
  3. Press "c" (Edit>Copy) and "v" (Edit>Paste).
  4. Drag the selection so that it is over the clown's right eye.
  5. Click outside the selection.
  6. The right eye and left eye should now be the same.
