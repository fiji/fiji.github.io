Comment 4 Curtis Rueden 2013-08-15 15:15:15 CDT 
As of today, we now have a dedicated Bio-Formats update site for daily builds. By enabling this update site, your Fiji will gain access to the latest Bio-Formats bugfixes, including the one which addresses this bug report.

To use it:

* Help > Update Fiji
* Click "Manage update sites"
* Check "Bio-Formats daily builds"
* Click "Close"

Five files should appear on the list with "Status/Action" of "Update it".

* Click "Apply changes"
* Restart Fiji
* Help > About Plugins > LOCI Plugins...

The dialog should now show a build date of "14 August 2013" or later.

Please let us know how it works for you!
