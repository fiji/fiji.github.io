Description Andrew Vaughan 2009-10-23 10:33:00 CDT 
The current (as of 23/10/09) version of Bio-Formats in FIJI doesn't sort the X, Y, Z and channels dimensions correctly for XZY stacks in Leica LIF files. These are where the fast scanning galvo stage insert is used to generate stacks in a plane orthogonal to the usual XYZ. Note that stable version 4 of Bio-Formats from June this year handles them correctly in Image J as I asked Melissa Linkert to fix this for me.
