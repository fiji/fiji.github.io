Compile the following example plugin somewhere within the ./plugins folder:

import ij.IJ;
import ij.plugin.PlugIn;

public class Example_PlugIn implements PlugIn {
	@Override
	public void run(String arg) {
		String test = IJ.getFilePath("Choose a file");
	}
}

Then run "Example Plugin" while running the script recorder. It will record something like:

run("Example PlugIn", "choose=/Users/eglinger/Desktop/test.tif choose=/Users/eglinger/Desktop/test.tif");

containing the parameter (in this case, "choose") twice.
