Comment 1 Curtis Rueden 2016-03-22 13:55:00 CDT 
Try unchecking the "Group files with similar names" option.

Also: did you place additional files into the same directory as the ZVI file? Bio-Formats takes the liberty of scanning the directory for other files named in a similar way. For example, if you had:

   /Users/jdoe/embryo1.zvi

And then later you added:

   /Users/jdoe/embryo2.zvi

Then Bio-Formats would notice the 1-2 pattern and want to import them together, at least when the "Group files with similar names" option is checked.
