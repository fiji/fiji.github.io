Comment 1 Wayne Rasband 2014-03-24 09:36:02 CDT 
There are two ways to change the width of a line selection. You can open the "Line Width" widget by double clicking on the line tool icon in the toolbar and use it to set the global line width. Or you can type 'y' (Edit>Selection>Properties) to open the "Properties' dialog, which allows you to set the width of the current line selection. As far as I know, it has never been possible to set the width of a line selection by double clicking on the line itself.
