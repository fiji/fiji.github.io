Comment 1 Wayne Rasband 2015-11-15 13:43:33 CST 
The scale bar is an overlay so you need to "flatten" the image before saving in TIFF format by pressing shift-f (Image>Overlay>Flatten). Or, when creating a scale bar, uncheck "Overlay" in the dialog box.
