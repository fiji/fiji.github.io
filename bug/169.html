Description Jan Eglinger 2010-05-17 19:25:23 CDT 
When you run "Linear Stack Alignment with SIFT" on a stack (with the default options, just clicking OK), the new image window is getting a z-slider with size=2 when the window is converted to a stack window.
This z-slider remains when the alignment is done, and doesn't update even when it is moved by dragging with the mouse, i.e. you seem to see only the first two slices. It only updates when scrolling with the mouse wheel, which is however not obvious to new users :)

(This has been tested on iMacs running MacOS 10.6.3 (Snow Leopard) and an updated Fiji version)
