Description Jan Eglinger 2012-08-21 19:19:26 CDT 
To reproduce, save the following script as Test_.js into <Fiji-path>/plugins

imp = IJ.getImage();
imp = new Duplicator().run(imp, 1, 1, 1, 1, 1, 1);
imp.show();

Then,
- refresh the menus (Help>Refresh Menus)
- start the Recorder (Plugins>Macros>Record...)
- switch to Record: JavaScript
- open any image
- run the script via the menu (Plugins>Test in my case)

Have a look at the recorder window, it wrongly records:
imp = new Duplicator().run(imp, 1, 1, 1, 1, 1, 1);

If you comment out line 2 in the script, the recorder correctly records:
IJ.run("Test ", "");

(When recording in Macro mode, everything seems ok.)
