Description Jan Eglinger 2013-09-12 04:48:21 CDT 
Since a recent Fiji update, opening image files from Finder or by dragging them to the dock does not work any more. Instead of opening the file, nothing happens (and if started from the console, no additional output is shown (after the initial INFO lines))

Opening via 'File > Open...' or by dragging to the Fiji main window works however.

(Tried on MacOS 10.6.8 with some lsm and png files on an up-to-date Fiji installation.)
Comment 1 Johannes Schindelin 2013-09-12 15:14:11 CDT 
Hey Jan, thanks for the bug report!

The problem was a recent change in scijava-common making work with SciJava contexts easier and my overzealous attempt to exploit that in ij-legacy.

The fix is in https://github.com/imagej/imagej/commit/651d8e7f47b11c7e96308fa91268f3dff73acd19 and I uploaded a new version of ij-legacy.jar with which the problem is fixed here.
Comment 2 Jan Eglinger 2013-09-13 06:47:45 CDT
