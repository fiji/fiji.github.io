Comment 2 Mark Hiner 2014-06-17 11:33:45 CDT 
I updated the summary of this ticket.

To clarify, the problem here is that we have an ImageJ 1.x TIFF with a G-R-B color table. When opened by SCIFIO, it displays as R-G-B.

Note that when opened via the Bio-Formats importer we get a grayscale image (still RGB if changed to composite or color mode via the Channels Tool).

The problem is that the color tables embedded in the ImageJ comment are not parsed properly: https://github.com/scifio/scifio/issues/202
